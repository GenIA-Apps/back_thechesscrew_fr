# back_thechesscrew_fr

Node.js + Express Backend API for ia_thechesscrew_fr

## 🚀 Quick Start

### Setup
```bash
npm install
```

### Development
```bash
npm run dev
```
Server runs on `http://localhost:3000`

### Production
```bash
npm start
```

## 🔌 API Endpoints

- `GET /api/health` - Health check
- `GET /api/hello` - Hello message with timestamp
- `GET /api/data` - Sample data endpoint

## 📝 Environment Variables

```
PORT=3000
NODE_ENV=development
```

## 🧪 Testing

```bash
npm test
```

## 📦 Technologies

- Node.js v22+
- Express.js
- CORS enabled
- dotenv for configuration
